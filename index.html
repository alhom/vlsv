VLSV

Parallel File Format with a Visualization Plugin

View project on GitHub

Welcome to VLSV Repository

VLSV file format was created for writing domain-decomposed unstructured meshes and associated variables efficiently in parallel using the message passing interface (MPI). Here are some features:

  • Data written in parallel using collective MPI calls
  • Plugin for parallel visualization using VisIt
  • Scales to tens of millions of cells per mesh
  • Multiple meshes per file
  • Supports multi-domain meshes, e.g., one mesh per MPI process
  • Support for many common mesh geometries, such as Cartesian, cylindrical, and spherical coordinate systems, and generic unstructured meshes
